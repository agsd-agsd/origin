    ```go
    // 基于 x = [0, 1, 2, 3, 4] 的一元线性回归斜率计算（简化 OLS）
    func calculateSlope(history []float64) float64 {
        n := 5
        if len(history) < n { return 0 }
        
        recent := history[len(history)-n:]
        var sumY, sumXY float64
        for i, y := range recent {
            sumY += y
            sumXY += float64(i) * y
        }
        return (5.0*sumXY - 10.0*sumY) / 50.0
    }
    ```

### 阶段 2：移植策略与业务逻辑 (预计 1.5 天)
目标：将 Python 中的边界更新（`update_rate_bounds`）和竞争调整规则移植为 Go 函数。

1.  **实现 `UpdateRateBounds` (边界管理)**：
    *   读取 `NetRevenueHistory` 和 `InvestedFundsHistory` 的最后 5 个元素，调用 `calculateSlope` 计算 `revenue_change` 和 `funds_change`。
    *   **成功/失败判定**：如果收益和资金都在增长（或持平），`ConsecutiveSuccess++`；如果大幅下跌，`ConsecutiveFailure++`。
    *   **动态调整上线**：若连续成功达到 5 次，提高 `HistoricalMaxSuccessfulRate`；若连续失败达到 2 次或资金单轮暴跌超过10%，则大幅下调该上限。
2.  **实现 `CalculateAdjustmentCoefficient` (竞争系数)**：
    *   定义寻找竞争对手资金最大值的接口。
    *   比较自身外部注资与最强竞争对手的外部注资，计算比值（1.2倍给 0.1 调幅，2倍以上给 0.2 调幅）。
3.  **合并策略得到 `Optimize` 方法**：
    *   计算本次的 Bounds（随迭代轮次 `Iteration` 下降收敛）。
    *   获取调整系数，判断当前比对手强还是弱。落后则降价，领先则温和提价。
    *   最终将计算出的新费率使用 `math.Max` 和 `math.Min` 限制在 `[MinFeeRate, UpperBound]` 之间返回。
